# Titanic Visualization Dashboard

## Data Science Internship - Task 4

## 📌 Project Overview
Created a mini visualization dashboard using the Titanic dataset to analyze survival patterns through data cleaning, feature engineering, and visualizations.

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Analysis & Visualizations
- Survival analysis by gender and passenger class
- Age distribution analysis
- Family size survival analysis
- Correlation heatmap
- Age group survival analysis

## 📈 Key Insights
- Female passengers had higher survival rates.
- First-class passengers had better survival chances.
- Family size and age influenced survival outcomes.
